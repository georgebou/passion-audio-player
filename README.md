# passion-audio-player
Passion Audio Player is a free, lightweight audio player for Windows with support for MP3, FLAC, OGG, WMA, APE, WAV and many more formats. Compatible with Winamp DSP plugins and Sonique visualization plugins. In development since 2002.

**Latest version: 5.0 (April 2019)**  
**Homepage: [passionplayer.com](https://www.passionplayer.com)**
